# Employee Management System - SQL Project

## Overview
A comprehensive Employee Management System database designed to handle employee records, departments, projects, and payroll management. This project demonstrates advanced SQL concepts including joins, subqueries, stored procedures, triggers, and data analysis.

## Database Schema
- **employees**: Employee personal and professional information
- **departments**: Department details and management
- **projects**: Project information and assignments
- **employee_projects**: Many-to-many relationship between employees and projects
- **salaries**: Employee salary history and payroll data

## Features
- Employee CRUD operations
- Department management
- Project assignment tracking
- Salary history and payroll calculations
- Advanced reporting and analytics
- Data integrity with constraints and triggers

## SQL Concepts Demonstrated
- Database design and normalization
- Complex JOIN operations
- Subqueries and CTEs
- Stored procedures and functions
- Triggers for data integrity
- Indexes for performance optimization
- Views for data abstraction
- Window functions for analytics

## Files Structure
- `schema.sql` - Database schema creation
- `sample_data.sql` - Sample data insertion
- `queries.sql` - Complex queries and reports
- `procedures.sql` - Stored procedures and functions
- `triggers.sql` - Database triggers
- `views.sql` - Database views

## How to Run
1. Execute `schema.sql` to create the database structure
2. Run `sample_data.sql` to populate with sample data
3. Use queries from other files for various operations
